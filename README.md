# SIS app
Program provide some portal and SIS services built using pure python using following libraries:
* Flet: User interface.
* Requests: comunication with websites.
* BeautifulSoup4: Scraping websites data.

# Services
* View Timetable
* View Registed Courses
* View Grades
* Calculate GPA
